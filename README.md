# Sistema de Controle de Qualidade Industrial (Python)

## Descrição

Este projeto simula um sistema de controle de qualidade em uma linha de produção industrial.

O sistema permite cadastrar peças, avaliar automaticamente se estão dentro dos padrões e organizar as peças aprovadas em caixas.

---

## Funcionalidades

- Cadastro de peças com:
  - ID
  - Peso
  - Cor
  - Comprimento

- Validação automática com critérios:
  - Peso entre 95g e 105g
  - Cor azul ou verde
  - Comprimento entre 10cm e 20cm

- Classificação:
  - Peças aprovadas
  - Peças reprovadas (com motivo)

- Organização:
  - Caixas com capacidade de 10 peças

- Relatórios:
  - Total de aprovadas
  - Total de reprovadas
  - Motivos de reprovação
  - Quantidade de caixas utilizadas
